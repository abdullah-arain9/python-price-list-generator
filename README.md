# Python Price List Generator

This project is a Python-based price list management system that allows users to manage categorized products and automatically generate a formatted price list image.

## Features

- Add new product categories
- Add products with prices
- Update product prices
- Delete products or categories
- Store product data using JSON
- Generate a formatted price list image
- Simple command-line interface

## Technologies Used

- Python
- Pandas
- Matplotlib
- JSON

## Installation

1. Clone the repository:

git clone https://github.com/abdullah-arain9/python-price-list-generator

2. Go to the project folder:

cd python-price-list-generator

3. Install required libraries:

pip install pandas matplotlib

## Usage

Run the Python file:

python price_list_generator.py

Then choose options from the menu:

1. Show Menu  
2. Add Category  
3. Add Product  
4. Update Price  
5. Delete Product  
6. Delete Category  
7. Generate Price List Image  
8. Exit

## Example Output

The program generates a formatted price list image containing categorized products and their prices.
